📄 BUILD PLAN SUMMARY

Product: UBS CV Builder (stand-alone)  |  Strategy: Template-First  |  Stack: SvelteKit, SQLite, Puppeteer, Tailwind  |  Approved: 2026-04-05 (KITT-amended v1.1)

Structured qualifications/CV production system using a fixed branded background template with text overlays. Background is a one-time master asset reused for all CVs. PDF fidelity is first-class. Approved CVs are immutable snapshots — edits fork a new draft. AI output is reviewable and non-authoritative.
