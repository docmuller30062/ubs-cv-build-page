📄 BUILD PLAN SUMMARY

Product: UBS CV Builder (stand-alone)  |  Strategy: Template-First  |  Stack: SvelteKit, SQLite, Puppeteer, Tailwind  |  Approved: 2026-04-05 (KITT-amended v1.1)

The UBS CV Builder is a structured qualifications/CV production system using a fixed branded background template with text overlays. Background is a one-time master asset reused for all CVs. Only text is imported from consultant documents. PDF fidelity is first-class. Approved CVs are immutable snapshots — edits fork a new draft. AI output is reviewable and non-authoritative.

Phase Structure

  • Phase 1: Template Contract + Renderer Proof (Steps 1-2)
  • Phase 2: Consultant Editor (Steps 3-4)
  • Phase 3: Workflow + Admin (Steps 5-7)
  • Phase 4: AI Assist (Steps 8-9)
  • Phase 5: Hardening + Integration Hook (Steps 10-11)
